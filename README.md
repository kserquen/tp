# tp

## Clone

`git clone https://github.com/kserquen/tp.git`.


## Install

`cd tp && npm install`.


## Configuring

`cd config`.

`mv config_example.json config.json`.

Modifica según tu configuración

## Development server

Ejecuta `npm start` en el servidor de desarrollo. Navega con la dirección `http://localhost:3000/`.

## Tests

Ejecuta `npm test`.
